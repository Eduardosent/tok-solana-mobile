# Tok Mobile App

Tok is an ecosystem and standard to create tokens, NFTs, and digital economies without coding. This repository contains the mobile application developed for the **Solana Seeker**.

## Project Development
- **Mobile App:** Flutter-based interface for seamless asset management.
- **Smart Contract:** The core engine (Rust/Anchor) development started on **February 28, 2026**, to power the ecosystem's logic.

## Smart Contract Repository
The custom smart contract for **Staking, Vesting, and the Launchpad DEX** is located here:
👉 [https://github.com/Eduardosent/tok-sol-contract](https://github.com/Eduardosent/tok-sol-contract)

## Key Features (v1 Devnet)
- **Factory:** Native SPL Token and NFT creation.
- **Tok Utils:** - **Staking:** Reward users for locking their assets.
  - **Vesting:** Manage distribution with cliff and intervals.
- **Launchpad DEX:** Permissionless marketplace for token presales.

## Vision
To establish a universal standard for **RWA Tokenization**, including Real Estate and Stock Markets, while providing SDKs for developers to integrate Tok as a service.
